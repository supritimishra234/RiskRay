RiskRay is a web application that helps users uncover hidden risks present in everyday information. Whether it's food ingredients, loan terms, insurance policies, or agricultural inputs, RiskRay analyzes the text and highlights potential health and financial concerns that people often overlook.
 
 Features:
-  Hidden risk detection from text
-  TF-IDF based keyword extraction
-  Health risk analysis , Financial risk analysis,  Insurance policy analysis
-  Agriculture-related risk detection
-  Risk score and impact visualization

Tech Stack:
 Frontend:-
- React
- HTML
- CSS
- JavaScript

 Backend:-
- Python
- Flask
- Flask-CORS

 NLP:-
- Scikit-Learn
- TF-IDF Vectorizer
